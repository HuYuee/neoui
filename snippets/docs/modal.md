# madal控件

`modal`控件分为三种：提示模态框、确认模态框和自定义模态框

# 如何使用

提示模态框与确认模态框通过js调用传参实现，
例如`u.messageDialog({ msg: "HELLO!!!", title: "测试提示", btnText: "OK!" })` 则是调用提示模态框的方法。
自定义模态框则是在html文件里可自定义内容
